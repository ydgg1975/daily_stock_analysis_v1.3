# New Plans

이 파일은 주식분석 에이전트 고도화 작업의 남은 계획만 유지한다.
구현이 완료된 항목은 위에서부터 하나씩 삭제한다.

## 1. 성과 추적과 백테스트

- 과거 리포트의 판단을 기록한다.
- 이후 1일, 1주, 1개월, 3개월 수익률을 추적한다.
- 벤치마크 대비 성과를 비교한다.
- confidence score별 성과를 분석한다.
- 리스크 경고가 실제 하락을 잘 포착했는지 평가한다.

## 2. 포트폴리오 분석

- 보유 종목과 비중을 입력하거나 저장할 수 있게 한다.
- 섹터, 국가, 통화 노출을 포트폴리오 단위로 분석한다.
- 상관관계 기반 분산도를 측정한다.
- 리밸런싱 제안을 생성한다.
- 신규 종목이 기존 포트폴리오에 미치는 영향을 분석한다.

## 3. 이벤트 모니터링

- 실적 발표 캘린더를 반영한다.
- 공시와 뉴스 변화를 모니터링한다.
- 가격 급등락과 거래량 이상을 감지한다.
- 기존 thesis가 깨지는 이벤트를 감지한다.
- 알림 우선순위를 분류한다.

## 4. 멀티모달 차트 분석

- 캔들 차트 이미지를 생성한다.
- 이동평균, 거래량, RSI, MACD를 시각화한다.
- 차트 패턴과 지지, 저항 후보를 분석한다.
- 수치 지표와 차트 해석이 충돌할 때 표시한다.

## 5. Paper Trading / 실행 준비

- paper trading 계좌와 거래 기록을 만든다.
- 주문 전 사용자 승인 플로우를 설계한다.
- 손실 한도, 종목별 최대 비중, 일일 거래 횟수 제한을 둔다.
- 거래 사유와 실행 결과를 기록한다.
- 브로커 API 추상화 계층을 준비한다.
