# 초보자용 클라이언트 설정 가이드

코드를 직접 수정하지 않고 클라이언트에서 분석을 시작하려는 사용자를 위한 안내입니다.

## 준비물

- 실행 가능한 클라이언트 또는 Web UI
- AI 모델 API 키
- 분석할 종목 코드

권장 종목 코드 형식:

```text
KR005930,CN600519,hk00700,AAPL
```

## 1. 클라이언트 실행

데스크톱 앱을 사용하는 경우 설치 후 실행합니다. Web UI를 사용하는 경우 서버를 먼저 시작합니다.

```bash
python main.py --serve
```

## 2. AI 모델 설정

설정 화면에서 API 키, API 주소, 모델명을 입력합니다.

OpenAI 호환 API 예시:

```text
API Key: your_api_key
Base URL: https://api.openai.com/v1
Model: gpt-4o-mini
```

입력 후 저장하고 연결 테스트를 실행합니다.

## 3. 종목 추가

종목 목록에 분석할 코드를 입력합니다.

```text
KR005930,AAPL
```

여러 종목은 쉼표로 구분합니다.

## 4. 뉴스 검색 설정

뉴스 검색 공급자는 필수는 아니지만 리포트 품질에 큰 영향을 줍니다. 한국 경제 뉴스와 증권사 리포트를 반영하려면 한국어 검색 결과가 잘 나오는 공급자를 연결하세요.

## 5. 첫 분석 실행

1. 홈 화면에서 종목 코드를 입력합니다.
2. 분석 버튼을 누릅니다.
3. 작업 상태가 완료될 때까지 기다립니다.
4. 히스토리 또는 리포트 화면에서 결과를 확인합니다.

## 자주 겪는 문제

### 모델 연결이 실패합니다

- API 키가 맞는지 확인합니다.
- API 주소에 `/v1`이 필요한지 확인합니다.
- 모델명이 공급자에서 실제로 제공되는지 확인합니다.

### 리포트가 비어 있습니다

- 종목 코드가 올바른지 확인합니다.
- 데이터 공급자 또는 뉴스 검색 공급자의 응답 상태를 확인합니다.
- 로그에 timeout 또는 rate limit 메시지가 있는지 확인합니다.

### 설정을 초기화하고 싶습니다

설정 백업을 먼저 내보낸 뒤 필요한 항목만 다시 입력하세요.
