# 설정 도움말

Web UI와 환경 변수에서 사용하는 주요 설정을 한국어로 정리합니다.

## 기본 설정

| 항목 | 설명 |
| --- | --- |
| 종목 목록 | 분석할 종목 코드 목록 |
| 실행 시간 | 예약 분석 실행 시간 |
| 리포트 언어 | 사용자에게 표시되는 리포트 언어 |
| 모델 공급자 | AI 분석에 사용할 모델 공급자 |
| 뉴스 검색 | 종목 뉴스와 경제 뉴스 검색 공급자 |

## 종목 코드

시장 충돌을 피하려면 접두사를 붙이는 방식을 권장합니다.

```text
KR005930,CN600519,hk00700,AAPL
```

## AI 설정

필수 항목:

- API 키
- API 기본 주소
- 모델명

모델 호출이 실패하면 모델명, endpoint, API 키 권한을 먼저 확인하세요.

## 알림 설정

하나 이상의 알림 채널을 설정할 수 있습니다. 채널별 상세 내용은 `docs/notifications.md`를 참고하세요.

## 저장 전 확인

- 필수 값이 비어 있지 않은지
- API 키가 화면이나 로그에 노출되지 않는지
- 숫자와 문자열 형식이 맞는지
- 변경 후 단일 종목 분석이 정상 동작하는지
